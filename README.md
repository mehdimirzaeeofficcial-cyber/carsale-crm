# 🚗 سیستم CRM فروش خودرو اقساطی

سیستم جامع مدیریت ارتباط با مشتری برای فروشگاه‌های خودرو با قابلیت فروش اقساطی

## ✨ ویژگی‌ها

- 👥 مدیریت مشتریان
- 🚗 کاتالوگ خودروها
- 📋 سیستم قراردادها
- 💰 پیگیری اقساط
- 📞 ارتباطات (تلفن، پیامک، واتساپ)
- 👨‍💼 مدیریت کارشناسان
- 📊 گزارش‌گیری
- ⚙️ تنظیمات شرکت
